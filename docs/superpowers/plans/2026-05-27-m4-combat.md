# 灵渊 M4 实施计划：战斗 / 怪物 / PvP

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use `- [ ]` checkboxes.

**Goal:** 让世界活起来 —— 动物乱跑可猎，夜里有怪，agent 之间能动手。

**Architecture:** 在 World 加 `creatures: BTreeMap<EntityKey, Creature>`；新动作 `Attack { target: AttackTarget }`；新 system `creature_ai` 每 tick 跑一遍；夜晚 spawn 怪物；攻击在 tick 边界结算，伤害用武器表 + 距离判定。

## 范围

**IN：**
- 3 种动物：rabbit（中立/逃跑，掉肉）、deer（中立群体/逃跑，掉肉+鹿茸）、wolf（群体主动夜出）
- 1 种怪：night_demon（黑石冢夜出，主动追击，掉 spirit_jade 概率掉落）
- attack 动作（target 可为 agent / creature_id）
- 武器伤害：拳=3、bamboo_spear=8、stone_axe=10、iron_sword=18（iron_sword 已在表里）
- 损耗 stamina
- 死亡掉落 raw_meat（动物）或 spirit_jade（怪）
- 夜晚 spawn：夜间 phase 时每 N tick 在玩家附近 spawn 1 wolf 或 night_demon

**OUT（推迟到后续）：**
- warmth 真实衰减
- sanity 衰减（怨魂凝视等）
- boss
- 季节差异化 spawn

## Task M4-1: CreatureKind + Creature

`crates/world/src/creature.rs`:

```rust
use serde::{Deserialize, Serialize};
use crate::{AgentId, ItemKind, TileCoord};

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum CreatureKind {
    Rabbit, Deer, Wolf, NightDemon,
}

impl CreatureKind {
    pub fn max_hp(self) -> i16 {
        match self {
            CreatureKind::Rabbit => 8,
            CreatureKind::Deer => 24,
            CreatureKind::Wolf => 30,
            CreatureKind::NightDemon => 50,
        }
    }
    pub fn attack(self) -> i16 {
        match self {
            CreatureKind::Rabbit | CreatureKind::Deer => 0,
            CreatureKind::Wolf => 8,
            CreatureKind::NightDemon => 12,
        }
    }
    pub fn vision(self) -> u16 { 5 }
    pub fn is_hostile(self) -> bool {
        matches!(self, CreatureKind::Wolf | CreatureKind::NightDemon)
    }
    pub fn loot(self) -> &'static [(ItemKind, u16)] {
        match self {
            CreatureKind::Rabbit => &[],  // M4 不引入 raw_meat item kind；放到 v1.1
            CreatureKind::Deer => &[],
            CreatureKind::Wolf => &[],
            CreatureKind::NightDemon => &[],
        }
    }
}

pub type CreatureId = u64;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Creature {
    pub id: CreatureId,
    pub kind: CreatureKind,
    pub hp: i16,
    pub pos: TileCoord,
    pub target: Option<AgentId>,
    pub last_action_tick: u64,
}

impl Creature {
    pub fn new(id: CreatureId, kind: CreatureKind, pos: TileCoord, tick: u64) -> Self {
        Self { id, kind, hp: kind.max_hp(), pos, target: None, last_action_tick: tick }
    }
}
```

注：M4 简化版不引入 RawMeat ItemKind；战斗后死亡只是 creature 消失。后续 M4.1 加 RawMeat。

## Task M4-4: resolve_attack + creature.attack_agent

在 world.rs：
- `creatures: BTreeMap<u64, Creature>` 字段
- bootstrap 初始化空
- `step` 里加 creature_ai system + 夜晚 spawn system
- `resolve_attack` 新方法
- agent 当前没有 weapon 槽 → 简化：找 inventory 里"最贵"武器（StoneAxe > BambooSpear > None）

```rust
fn agent_best_weapon(a: &Agent) -> Option<ItemKind> {
    if a.inventory.count(ItemKind::StoneAxe) > 0 { Some(ItemKind::StoneAxe) }
    else if a.inventory.count(ItemKind::BambooSpear) > 0 { Some(ItemKind::BambooSpear) }
    else { None }
}

fn resolve_attack(&mut self, aid: AgentId, target: AttackTarget) {
    let Some(agent) = self.agents.get(&aid) else { return; };
    let attacker_pos = agent.pos;
    if agent.status.stamina < 5 {
        self.pending_events.push(TickEvent::AgentMoveFailed { agent: aid, reason: "exhausted".into() });
        return;
    }
    let weapon = agent_best_weapon(agent);
    let dmg = crate::combat::resolve_attack_damage(weapon);
    match target {
        AttackTarget::Agent(target_id) => {
            let Some(target) = self.agents.get(&target_id) else { return; };
            if attacker_pos.manhattan(target.pos) > 1 { return; }
            let target_pos = target.pos;
            let target = self.agents.get_mut(&target_id).unwrap();
            target.status.hp = (target.status.hp - dmg).max(0);
            self.pending_events.push(TickEvent::AgentAttackedAgent {
                attacker: aid.clone(), target: target_id, damage: dmg, weapon: weapon.map(|w| format!("{:?}", w)),
            });
        }
        AttackTarget::Creature(cid) => {
            let Some(creature) = self.creatures.get(&cid) else { return; };
            if attacker_pos.manhattan(creature.pos) > 1 { return; }
            let creature = self.creatures.get_mut(&cid).unwrap();
            creature.hp = (creature.hp - dmg).max(0);
            self.pending_events.push(TickEvent::AgentAttackedCreature {
                attacker: aid.clone(), creature_id: cid, damage: dmg,
            });
            if creature.hp <= 0 {
                let pos = creature.pos;
                let kind = creature.kind;
                self.creatures.remove(&cid);
                self.pending_events.push(TickEvent::CreatureKilled { id: cid, kind, at: pos });
            }
        }
    }
    let attacker = self.agents.get_mut(&aid).unwrap();
    attacker.status.stamina = (attacker.status.stamina - 5).max(0);
    attacker.last_action_tick = self.clock.tick;
}
```

## Task M4-5: systems/creature_ai.rs

每 tick 跑：动物逃跑 / 怪物追击。简单实现：
- rabbit/deer: 看到 agent 8 格内 → 往反方向走 1 步
- wolf/night_demon: 在 vision 内找最近 agent → 朝它走 1 步，到 1 格内就攻击

```rust
use crate::{
    agent::{Agent, AgentId, AgentState, AgentStatus},
    coord::{Direction, TileCoord},
    creature::{Creature, CreatureId, CreatureKind},
    event::TickEvent,
    grid::Grid,
    tile::Tile,
};
use std::collections::BTreeMap;

pub fn step_creatures(
    tick: u64,
    grid: &Grid<Tile>,
    agents: &mut BTreeMap<AgentId, Agent>,
    creatures: &mut BTreeMap<CreatureId, Creature>,
) -> Vec<TickEvent> {
    let mut events = Vec::new();
    let ids: Vec<CreatureId> = creatures.keys().copied().collect();
    for cid in ids {
        let Some(c) = creatures.get(&cid).cloned() else { continue; };
        let vision = c.kind.vision();
        let nearest = agents
            .iter()
            .filter(|(_, a)| matches!(a.state, AgentState::Alive))
            .filter(|(_, a)| a.pos.manhattan(c.pos) <= vision)
            .min_by_key(|(_, a)| a.pos.manhattan(c.pos));
        let Some((aid, agent)) = nearest else { continue; };
        let dx = (agent.pos.x - c.pos.x).signum() as i16;
        let dy = (agent.pos.y - c.pos.y).signum() as i16;
        if c.kind.is_hostile() {
            // 临近就攻击
            if c.pos.manhattan(agent.pos) <= 1 {
                let aid_owned = aid.clone();
                let creature_attack = c.kind.attack();
                if let Some(a_mut) = agents.get_mut(&aid_owned) {
                    a_mut.status.hp = (a_mut.status.hp - creature_attack).max(0);
                }
                events.push(TickEvent::CreatureAttackedAgent {
                    creature_id: cid,
                    creature_kind: c.kind,
                    target: aid_owned,
                    damage: creature_attack,
                });
                continue;
            }
            // 朝玩家走
            let new_pos = pick_step_towards(grid, agents, creatures, c.pos, dx, dy);
            if let Some(p) = new_pos {
                creatures.get_mut(&cid).unwrap().pos = p;
            }
        } else {
            // 逃跑：往反方向
            let new_pos = pick_step_towards(grid, agents, creatures, c.pos, -dx, -dy);
            if let Some(p) = new_pos {
                creatures.get_mut(&cid).unwrap().pos = p;
            }
        }
    }
    let _ = tick;
    events
}

fn pick_step_towards(
    grid: &Grid<Tile>,
    agents: &BTreeMap<AgentId, Agent>,
    creatures: &BTreeMap<CreatureId, Creature>,
    from: TileCoord,
    dx: i16,
    dy: i16,
) -> Option<TileCoord> {
    let dirs: [(i16, i16); 4] = [(dx, 0), (0, dy), (dx, dy), (-dx, dy)];
    for (mx, my) in dirs {
        if mx == 0 && my == 0 { continue; }
        let target = TileCoord::new(from.x + mx.signum(), from.y + my.signum());
        if !grid.get(target).map(|t| t.is_walkable()).unwrap_or(false) { continue; }
        if agents.values().any(|a| a.pos == target) { continue; }
        if creatures.values().any(|c| c.pos == target) { continue; }
        return Some(target);
    }
    None
}
```

## Task M4-6: systems/spawn.rs

夜晚每 8 tick 尝试在距 agent 8~12 格处 spawn 1 wolf 或 night_demon。

```rust
use crate::{
    agent::{Agent, AgentId, AgentState},
    coord::TileCoord,
    creature::{Creature, CreatureId, CreatureKind},
    grid::Grid,
    tile::Tile,
};
use std::collections::BTreeMap;

pub fn step_spawn(
    tick: u64,
    seed: u64,
    is_night: bool,
    grid: &Grid<Tile>,
    agents: &BTreeMap<AgentId, Agent>,
    creatures: &mut BTreeMap<CreatureId, Creature>,
    next_creature_id: &mut CreatureId,
) -> Vec<crate::event::TickEvent> {
    use rand::{Rng, SeedableRng};
    let mut out = Vec::new();
    if !is_night || tick % 8 != 0 { return out; }
    // 总怪上限 32，保护服务器
    if creatures.values().filter(|c| c.kind.is_hostile()).count() >= 32 { return out; }
    let mut rng = rand_chacha::ChaCha8Rng::seed_from_u64(seed ^ tick);
    let alive: Vec<&Agent> = agents.values().filter(|a| matches!(a.state, AgentState::Alive)).collect();
    if alive.is_empty() { return out; }
    let center = alive[rng.gen_range(0..alive.len())].pos;
    for _ in 0..20 {
        let dx = rng.gen_range(-12..=12);
        let dy = rng.gen_range(-12..=12);
        if dx.abs() + dy.abs() < 6 { continue; }
        let p = TileCoord::new(center.x + dx, center.y + dy);
        let walkable = grid.get(p).map(|t| t.is_walkable()).unwrap_or(false);
        if !walkable { continue; }
        if agents.values().any(|a| a.pos == p) { continue; }
        if creatures.values().any(|c| c.pos == p) { continue; }
        let kind = if rng.gen_bool(0.65) { CreatureKind::Wolf } else { CreatureKind::NightDemon };
        *next_creature_id += 1;
        let id = *next_creature_id;
        creatures.insert(id, Creature::new(id, kind, p, tick));
        out.push(crate::event::TickEvent::CreatureSpawned { id, kind, at: p });
        break;
    }
    out
}
```
